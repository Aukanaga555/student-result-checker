# Student Result Checker

A web application built with Django and Python that allows teachers to add student marks and students to check their results.

## Technologies Used
- Python
- Django
- SQLite
- HTML, CSS

## Features
- Add student details and marks
- Search result by roll number
- Pass/Fail automatically calculated

## How to Run
1. Clone the repository
2. Install Django: pip install django
3. Run migrations: py manage.py migrate
4. Start server: py manage.py runserver
5. Open: http://127.0.0.1:8000/add/
